# Vibe Coding Playground — Project Overview

## Что это?
Одностраничный showcase-сайт, демонстрирующий возможности современной веб-разработки с помощью AI-инструментов (Claude + Cursor). Каждая секция — отдельная визуальная история, показывающая конкретную технику или приём.

## Цель
Вдохновить людей на изучение вайбкодинга. Показать, что за 1 день с AI-ассистентами можно создать профессиональный, визуально впечатляющий сайт.

## Целевая аудитория
- Новички, которые хотят научиться современной веб-разработке
- Ученики курса по вайбкодингу
- Опытные разработчики, ищущие вдохновение
- Все, кто интересуется AI-assisted разработкой

## Секции сайта
1. **Hero** — первое впечатление, анимированный фон, заголовок
2. **Animations** — stagger animations, scroll-triggered reveals, hover effects
3. **Interactivity** — конфетти, морфинг кнопок, слайдеры
4. **Visual Effects** — glassmorphism, glow, градиенты, тени
5. **Responsive Design** — демонстрация адаптивности с интерактивным переключателем
6. **Footer** — призыв к действию, ссылки

## Ключевые принципы
- Каждый блок — самодостаточный компонент
- Dark theme по умолчанию + переключатель в light
- Мобильный first
- Производительность: анимации через CSS transforms, не layout properties
- Accessibility: aria-labels, focus states, reduced-motion support
